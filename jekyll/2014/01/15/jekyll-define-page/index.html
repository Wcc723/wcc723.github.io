еңЁJekyllдёӯиӘҝж•ҙеұ¬ж–јиҮӘе·ұзҡ„Template

2014 е№ҙ 01 жңҲ 15 ж—Ҙ В· 3 еҲҶйҗҳй–ұи®Җ В· By Wang Casper

жңғз”ЁjekyllжңүдёҖйғЁеҲҶжҳҜжғіиҰҒиҮӘе·ұиЁӯиЁҲзүҲеһӢпјҢжҲ‘жүҖж“…й•·зҡ„жҳҜиҰ–иҰәиЁӯиЁҲгҖҒHTMLгҖҒCSSпјҢжҲ‘е°ұз”ЁйҖҷе№ҫеҖӢжҠҖиғҪиЁӯиЁҲиҮӘе·ұзҡ„зүҲеһӢпјҢз•¶з„¶йҖҷдёҰдёҚжҳҜйӮЈйәјзҡ„дёҖе…©зҜҮе°ұеҸҜд»Ҙд»Ӣзҙ№е®ҢгҖӮжүҖд»ҘйҖҷзҜҮжңғиӘӘжҳҺдёҖеҖӢзҜ„дҫӢпјҢд»Ӣзҙ№Jekyllдёӯзҡ„зүҲеһӢйҒӢдҪңйӮҸијҜпјҢжңүиҲҲи¶Јзҡ„дҪҝз”ЁиҖ…еҸҜд»Ҙдҫқж“ҡйҖҷжҰӮеҝөиЁӯиЁҲеұ¬ж–јиҮӘе·ұзҡ„blogгҖӮ

жң¬зҜҮйҮҚй»һ

иӘҝж•ҙjekyll bootstrapзҡ„зүҲеһӢпјҢеҠ е…Ҙж–Үз« ж‘ҳиҰҒж–јйҰ–й ҒпјҢи—үжӯӨдәҶи§Јjekyll bootstrapзҡ„йҒӢдҪңж–№ејҸгҖӮ

еҸғиҖғпјҡhttp://truongtx.me/tags.html#jekyll-ref

жіЁж„Ҹ

з”ұж–јjekyllе…§дёҚиғҪйҖЈзәҢијёе…Ҙе…©еҖӢеӨ§жӢ¬иҷҹ жҲ–жҳҜ еӨ§жӢ¬иҷҹ+зҷҫеҲҶжҜ”пјҢжүҖд»Ҙзӯүзӯүзҡ„еӨ§жӢ¬иҷҹдёӯй–“йғҪжңғжҸ’е…Ҙз©әзҷҪйҡ”й–ӢпјҢCopyжҷӮиЁҳеҫ—еҲӘйҷӨвҖҰ.гҖӮ

{ { content } }
 ^	         ^  //иЁҳеҫ—ж¶ҲйҷӨ..

{ % include JB/setup %}
 ^	            //иЁҳеҫ—ж¶ҲйҷӨ..

Jekyll bootstrapжһ¶ж§Ӣ

йҰ–е…ҲжҲ‘еҖ‘еҶҚдҫҶзңӢдёҖдёӢз¶Із«ҷжһ¶ж§ӢпјҢйҖҷйӮҠеҲ—еҮәеҸҜиғҪжңғдҝ®ж”№еҲ°зҡ„жӘ”жЎҲгҖӮ

|-- index.html   #з¶Із«ҷйҰ–й Ғ
|-- _config.yml  #з¶Із«ҷиЁӯе®ҡжӘ”
|
|---/_includes  #з¶Із«ҷзөҗж§ӢиіҮж–ҷеӨҫ
|	|--/JB      #Jekyll-bootstrap жЁЎзө„
|	|--/themes  #Jekyll-bootstrap й җиЁӯзүҲеһӢ
|
|---/_layouts   #зүҲеһӢ
|
|---/_posts     #ж–Үз« иіҮж–ҷеӨҫ
|
|---/assets     #CSS,JS иіҮж–ҷеӨҫ

еүҚе№ҫзҜҮжңүд»Ӣзҙ№_config.ymlпјҢд»ҠеӨ©иҰҒд»Ӣзҙ№жҖҺйәјдҝ®ж”№index.htmlзҡ„зүҲеһӢгҖӮ

еңЁеӮізөұJekyllжһ¶ж§Ӣдёӯз¶Із«ҷйҰ–й Ғзҡ„index.htmlеҸҜд»ҘйҖҸйҒҺиЁӯе®ҡжӘ”еҺ»жҠ“еҸ–_layoutsе…§зҡ„зүҲеһӢпјҢindex.htmlиЁӯе®ҡеҸғиҖғеҰӮдёӢпјҡ

---
layout: default
---

иҖҢеңЁJekyll bootstrapе…§зҡ„_layoutsжңғеҶҚеҺ»ијүе…Ҙеұ¬ж–је®ғе°Ҳжңүзҡ„зүҲеһӢпјҢи·Ҝеҫ‘е°ұж”ҫеңЁ_includes/themes/twitter/е…§пјҢиҖҢеңЁlayoutе…§зҡ„default.htmlеҺҹе§ӢзўјеҰӮдёӢпјҡ

---
theme :
  name : twitter
---
{ % include JB/setup %}
{ % include themes/twitter/default.html %}

//еүҚ4иЎҢзҡ„жҳҜиЁӯе®ҡCSSзҡ„и·Ҝеҫ‘
//еҫҢ2иЎҢжҳҜеңЁијүе…Ҙinclude/themes/twitter/е…§зҡ„default.html

иҖҢжңҖеҫҢзҡ„include/themes/twitter/default.htmlзҡ„зөҗж§Ӣе°ұеғҸжҳҜдёҖиҲ¬зҡ„htmlпјҢиҖҢд»ҘдёҠзҡ„е…§е®№жңғиў«ж”ҫеңЁdefault.htmlзҡ„{ { content } }е…§гҖӮ

default.htmlзҡ„еҺҹе§Ӣзўј

иҖҢд»ҘдёҠдёүеҖӢи«ӢеҸғиҖғйҖҷејөең–пјҢжңҖе·Ұж–№зҡ„жҳҜи·ҹзӣ®йҢ„зҡ„index.htmlпјҢйҖҸйҒҺдёҖеұӨдёҖеұӨзҡ„еҺ»еҸ–зҡ„Jekyll bootstrapзүҲеһӢгҖӮ

й–Ӣе§Ӣ

дёҠйқўзҡ„д»Ӣзҙ№жңүй»һжҠҪиұЎпјҢжҺҘдёӢдҫҶе°ұдҫҶеҜҰйҡӣж“ҚдҪңдёҖж¬ЎеҸҜиғҪжңғжӣҙдәҶи§ЈпјҢи«ӢжҢүз…§жӯҘй©ҹж…ўж…ўең°еҒҡдёӢеҺ»гҖӮ

жӯҘй©ҹдёҖ

йҰ–е…Ҳжү“й–Ӣinclude/themes/twitter/иіҮж–ҷеӨҫпјҢдёҰдё”еүөдёҖеҖӢindex.htmlжӘ”жЎҲгҖӮ

иЈЎйқўзЁӢејҸзўји«Ӣијёе…Ҙд»ҘдёӢ(е°ұйҖҷйәјдёүиЎҢпјҢеү©дёӢзҡ„д»–жңғеҺ»жҠ“еҗҢеұӨзҡ„default.html)пјҡ

<div class="content">
	{ { content } }
</div>

жӯҘй©ҹдәҢ

жү“й–Ӣ_layoutsзҡ„иіҮж–ҷеӨҫпјҢдёҰдё”иӨҮиЈҪpost.htmlж–јеҺҹдҪҚзҪ®пјҢдёҰдё”дҝ®ж”№жӘ”еҗҚзӮәindex.htmlгҖӮ

жү“й–Ӣindex.htmlдҝ®ж”№д»ҘдёӢзЁӢејҸзўјгҖӮ

include themes/twitter/post.html

дҝ®ж”№жҲҗ
include themes/twitter/index.html

жӯҘй©ҹдёү

жү“й–Ӣи·ҹзӣ®йҢ„зҡ„index.htmlпјҢиІјдёӢд»ҘдёӢзЁӢејҸзўјгҖӮ

зЁӢејҸзўјпјҡhttps://github.com/wcc723demo/wcc723demo.github.io/blob/master/index.html

жҺҘдёӢдҫҶеҲ©з”Ёе…ҲеүҚд»Ӣзҙ№зҡ„Ruby jekyll serverдҫҶй җиҰҪзңӢзңӢз¶Ій ҒпјҢжҮүи©Іжңғеҫ—еҲ°йҖҷжЁЈзҡ„зөҗжһңпјҢд»–жҠҠзӣ®еүҚй Ғйқўзҡ„еүҚ100еӯ—ж“·еҸ–дәҶеҮәдҫҶгҖӮ

дёҠйқўйӮЈдёҖж®өзҡ„зЁӢејҸзўјзҡ„е…§е®№еҰӮдёӢ(з…©и«ӢиҮӘеӢ•иЈңи…ҰеӨ§жӢ¬иҷҹ)пјҡ

for post in site.posts limit:10
//жң¬з«ҷзҡ„10зӯҶж–Үз« 

post.title
//зӯүзӯүжңғз”ЁдҫҶж”ҫзҪ®дёӯж–ҮжЁҷйЎҢ

if post.content contains "<!-- more -->"
//еҰӮжһңе…§е®№еҢ…еҗ«"<!-- more -->" е°ұж“·еҸ–еҲ°"<!-- more -->"
//еҰӮжһңжІ’жңүе°ұж“·еҸ–100еӯ—

endfor,endif
