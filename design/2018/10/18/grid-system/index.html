960gs

з•¶з„¶з¶Ій ҒиЁӯиЁҲд№ҹдёҚдҫӢеӨ–пјҢжҲ‘еҖ‘йҖҡеёёзЁұзӮә Grid SystemгҖӮеңЁ RWD зӣӣиЎҢиө·дҫҶд№ӢеүҚжңҖи‘—еҗҚзҡ„з¶Іж јзі»зөұзЁұзӮә 960 Grid SystemпјҢдё»иҰҒжҳҜеӣ зӮәж—©жңҹзҡ„йӣ»и…Ұиһўе№•еҜ¬еәҰзҙ„зӮә 1024пјҢжүЈйҷӨзҖҸиҰҪеҷЁзҡ„жҚІи»ёеҸҠйӮҠжЎҶпјҢ960 еүҮжҳҜиЁұеӨҡж•ёеҖјзҡ„жңҖе°Ҹе…¬еҖҚж•ёпјҢжүҖд»Ҙиў«е»Јжіӣзҡ„иЁҺи«–еҸҠдҪҝз”ЁпјҢзӣҙеҲ°зҸҫеңЁиЁұеӨҡ CSS жЎҶжһ¶д№ҹжҳҜжңү 960 зҡ„иЁӯе®ҡеҖјеңЁе…§гҖӮ


https://960.gs/demo.html

960 Grid System зҡ„жҰӮеҝөжҳҜе°Ү 960 еҚҖеҲҶзӮә 12 ж¬„пјҲд№ҹеҸҜиЁӯе®ҡзӮә 16, 24пјҢдҫқж“ҡйңҖжұӮдёҚеҗҢжңғжңүдёҚеҗҢзҡ„ж¬„дҪҚж•ёпјүпјҢжҺҘдёӢдҫҶз¶Ій Ғдёӯзҡ„е…ғзҙ е°ұдҫқж“ҡж¬„дҪҚж•ёдҫҶжҺ’еҲ—е…§е®№пјҢеңЁ 960gs зҡ„з¶Ій Ғдёӯд№ҹжңүжҸҗдҫӣиЁұеӨҡзҜ„дҫӢжҳҜдҫқж“ҡйҖҷеҖӢжҰӮеҝөжүҖеҹ·иЎҢзҡ„гҖӮ

йӣ–з„¶зҜ„дҫӢдёӯзҡ„з¶Ій ҒйғҪжңүдәӣе№ҙд»ЈпјҢеңЁз¶Іж јзі»зөұзҡ„еј•е°ҺдёӢе…·жңүеӣәе®ҡзҡ„ж¬„еҜ¬еҸҠй–“йҡ”пјҢиіҮиЁҠзҡ„е‘ҲзҸҫдҫқз„¶ж•ҙйҪҠгҖҒжё…жҘҡпјҢдёҰдёҚжңғж„ҹеҸ—еҲ°йӣңдәӮйӣЈд»Ҙй–ұи®ҖгҖӮ

еҗҚи©һи§ЈйҮӢ

д»Ӣзҙ№ Grid System еҗҚи©һжҷӮпјҢжҲ‘е–ңжӯЎдҪҝз”Ё Sketch зҡ„е·Ҙе…·пјҢе®ғеҲ—еҮәжҲ‘еҖ‘еңЁдҪҝз”Ё Grid System зҡ„еёёз”ЁиЁӯе®ҡгҖӮ

Total Width: зёҪеҜ¬пјҢз¶Ій Ғдё»иҰҒе…§е®№е‘ҲзҸҫзҡ„зҜ„еңҚ
Offset: жҺЁз§»пјҢйҖҷжҳҜеұ¬ж–ј Sketch з•«еёғзҡ„иЁӯе®ҡеҸҜд»Ҙз„ЎиҰ–
Number of Column: зёҪж¬„дҪҚж•ёпјҢ12 жҳҜеёёз”Ёзҡ„ж•ёеҖјпјҢ16гҖҒ24 д№ҹжҳҜиЁұеӨҡй–ӢзҷјиҖ…дҪҝз”ЁпјҢйҖҷдәӣж•ёеҖјеҗҢжҷӮжҳҜиЁұеӨҡж•ёеҖјзҡ„жңҖе°Ҹе…¬еҖҚж•ёгҖӮ
Gutter Width: й–“йҡ”еҜ¬еәҰпјҢж¬„иҲҮж¬„зҡ„й–“и·қгҖӮ
Gutter on outside: еӨ–йғЁй–“и·қпјҢиЁӯе®ҡзҡ„й–“йҡ”еҜ¬еәҰжҳҜеҗҰиҰҒеҠ еңЁеӨ–еұӨгҖӮ
Column: е–®дёҖж¬„зҡ„еҜ¬еәҰпјҢзёҪж¬„дҪҚж•ёгҖҒй–“йҡ”еҜ¬еәҰеҸҠе–®дёҖж¬„зҡ„еҜ¬еәҰж•ёеҖјжңғжңүйҖЈеё¶зҡ„й—ңдҝӮгҖӮ

еҜҰйҡӣзҡ„зөҗжһңжңғеҰӮдёӢпјҢ960 жҳҜдёҖй–Ӣе§ӢиЁӯе®ҡзҡ„зёҪеҜ¬еәҰпјҢдҪҶеҰӮжһңжңүеҠ дёҠ Gutter on outside е°ұжңғе°Ү (й–“йҡ”еҜ¬еәҰ / 2) еҠ еҲ°з•«йқўзҡ„е·ҰеҸіеӨ–еұӨдёҠпјҢGutter on outside зҡ„з”ЁйҖ”жҳҜйҒҝе…Қз¶Ій Ғдё»иҰҒе…§е®№йҒҺж–јиІјиҝ‘зҖҸиҰҪеҷЁйӮҠз·ЈпјҢжүҖд»Ҙжңғй җе…Ҳдҝқз•ҷдёҖдәӣеҜ¬еәҰпјҢдҪҶзӣҙжҺҘдҝқз•ҷдёҖеҖӢ й–“йҡ”еҜ¬еәҰ жңғйЎҜеҫ—йҒҺеҜ¬гҖӮ

дёҖеҖӢж¬„зҡ„еҜ¬еәҰжҳҜ 60pxпјҢдҪҶе…©еҖӢж¬„дҪҚж•ёзҡ„еҜ¬еәҰдёҰйқһжҳҜ 120pxпјҢжңғеҶҚеҠ дёҠе…©иҖ…д№Ӣй–“зҡ„ж¬„дҪҚй–“йҡ”пјҢжүҖд»Ҙж•ёеҖјжңғжҳҜ 60 + 60 + 20 = 140pxгҖӮ
ж¬„дҪҚеҜ¬е…¬ејҸпјҡж¬„еҜ¬ * ж¬„дҪҚж•ё + (ж¬„дҪҚж•ё - 1)

йҷӨжӯӨд№ӢеӨ–жҜҸеҖӢж¬„еҜ¬зҡ„ж•ёеҖјд№ҹжҳҜеҸ—еҲ°гҖҢзёҪж¬„дҪҚж•ёгҖҒй–“йҡ”еҜ¬еәҰгҖҚзҡ„еҪұйҹҝпјҢеҜҰйҡӣдёҠжңғе»әиӯ°иҮӘиЎҢз№ӘиЈҪзңӢзңӢпјҢжҗӯй…Қе…¬ејҸдҫҶдәҶи§ЈйҖҷдәӣз”ўз”ҹзҡ„еҺҹзҗҶгҖӮ
е–®дёҖж¬„еҜ¬е…¬ејҸпјҡ(зёҪеҜ¬еәҰ - Gutter on outside) - ж¬„дҪҚеҜ¬еәҰ * (ж¬„ж•ё - 1)) / ж¬„ж•ё
зҜ„дҫӢпјҡ((960 - 20) - 20 * 11) / 12

д»ҘдёҠйғҪжҳҜи§ҖеҝөпјҢеҰӮжһңдәҶи§Јзҡ„и©ұе…¶еҜҰйҖҸйҒҺе·Ҙе…·з”ўз”ҹеҚіеҸҜпјҢдёҰдёҚйңҖиҰҒиЁҳжҶ¶йҖҷдәӣе…¬ејҸгҖӮ

жҺҘдёӢдҫҶеӣһеҲ°йҖҷеҖӢз¶Ій ҒдёҠпјҢзӣёдҝЎдҪ е°ұиғҪзҗҶи§ЈйҖҷдәӣж•ёеҖјзҡ„иЁӯе®ҡеҲҶеҲҘеҢ…еҗ«е“Әдәӣе…§е®№пјҢи©Ұи‘—жҖқиҖғпјҡеҒҮиЁӯдёӢж–№з¶Ій ҒзӮә 960pxгҖҒж¬„еҜ¬ 60pxгҖҒй–“и·қ 20pxпјҢдёҰдё”еҢ…еҗ«еӨ–йғЁй–“и·қзҡ„жғ…жіҒдёӢпјҢи«Ӣе•ҸеӣӣзұғеҜ¬еәҰзҡ„е…§е®№зҙ„дҪ”еӨҡеҜ¬пјҹ

еҲ°йҖҷйғЁеҲҶзӣёдҝЎдҪ е°Қж–јеҹәжң¬з¶Іж јзі»зөұжңүеҹәжң¬еҫ—дәҶи§ЈпјҢйӣ–з„¶еңЁиЁӯиЁҲжҷӮдёҚжңғдёҖж јдёҖж јеҺ»ж•ёеӨҡе°‘еҜ¬еәҰпјҲйҖҡеёёжҳҜзӣҙжҺҘжӢүж¬„дҪҚж•ёпјүпјҢдёҚйҒҺеңЁе…·еӮҷд»ҘдёҠи§ҖеҝөеҫҢпјҢжҲ‘еҖ‘еҶҚдҫҶд»Ӣзҙ№жҷӮжҡ«жҷӮжҲ‘еҖ‘жңғеҰӮдҪ•зҡ„йҒӢз”ЁйҖҷдәӣи§ҖеҝөгҖӮ
