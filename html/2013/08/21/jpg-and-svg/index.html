image與SVG的運用

來源 : http://lynn.ru/examples/svg/en.html

image

image和img是非常相似的，大部分瀏覽器都可以用image取代img replace

以上的原始碼

<svg>
<image xlink:href="images/20130821.svg" src="images/20130821.png" width="75" height="75"/>
</svg>

如果在支持SVG的瀏覽器會看到這樣

<image xlink:href="images/20130821.svg" width="75" height="75"/>

如果是不支持SVG的瀏覽器會顯示如下

<image src="images/20130821.png" width="75" height="75"/>

對於有些很簡單的圖像，她可能不需要獨立的SVG檔案，可以寫成這樣。

<svg height="16" width="16">
<path d="M5 1v14l9-7"> //SVG code
<image src="next.png"> //替代PNG
</svg>
